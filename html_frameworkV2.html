Welcome to Nomeclature Review Game!

Do you have want it takes to name any types of molecules covered in NE 122?

If so, this is the game for you!

The aim of the game is to catch the right molecule names in the little basket, the basket

You will be given few minutes before each level to see the images of each molecule

Then the game starts with you having 5 hearts and names of molcules dropping down from your screen

Each time you either catch the wrong answer or drop the correct one, you will lose one heart

The game will continue until the user loses all of their hearts or the level is completed

The point of this game is to improve on your skill in naming molecules from alkanes to cycloalkanes

However the main reason of this game is to have fun!!!
